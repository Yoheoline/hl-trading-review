## 自動切り替えロジック（案）

```
相場環境検出
    ↓
┌─────────────┬──────────────┬─────────────┐
│  高ボラ     │  中ボラ      │  低ボラ     │
│  +レンジ    │  +トレンド   │  +レンジ    │
│             │              │             │
│ スキャル    │  デイトレ    │  スイング   │
│ BB+Volume   │  MA Cross    │  RSI+Mom    │
│ TP:1%       │  TP:1.5%     │  TP:2%      │
└─────────────┴──────────────┴─────────────┘
```

### 環境検出指標
- **ATR**: ボラティリティ測定
- **ADX**: トレンド強度
- **BB幅**: バンド収縮/拡大

### 切り替えルール
1. ATR > 平均1.5倍 → スキャルピングモード
2. ADX > 25 + トレンド明確 → デイトレモード
3. ATR < 平均0.7倍 + レンジ → スイングモード
