# Trading Plan

各トレーダーが毎回チェックする計画ファイル。

## フォーマット

```markdown
- [ ] [条件] → [アクション]（Account X, size: X.XXX）
  - 根拠: [なぜこの計画を立てたか]
  - 作成: [日時] by [誰]
  - 有効期限: [いつまで有効か]
```

**例:**
```markdown
- [ ] BTC 84000以下 → LONG（Account 1, size: 0.005）
  - 根拠: 日足サポート + RSI売られすぎ圏
  - 作成: 1/30 01:00 by Swing Trader
  - 有効期限: 1/30 中

- [ ] 現ポジ +3%到達 → 半分利確（Account 4）
  - 根拠: 短期は早めに利確
  - 作成: 1/30 00:50 by Ultra Scalper
  - 有効期限: ポジション決済まで
```
